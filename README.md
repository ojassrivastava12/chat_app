# chat_app
Its just a hobby project, trying to learn and understand flask and socket io <br>

I have used Python 3.14.5, Flask 3.1.3 and Flask-SocketIO 5.6.1 for this project<br>

Features:<br>
1. one-on-one live chatting
2. user friendly
3. no login or sign up required
4. Supports multiple rooms at a single time
