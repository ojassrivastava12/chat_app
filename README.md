# chat_app
A lightweight real-time chat application built with Flask and Flask-SocketIO. Users can create chat rooms, join existing rooms using a room code, and communicate instantly with other participants. <br>

I have used Python 3.14.5, Flask 3.1.3 and Flask-SocketIO 5.6.1 for this project<br>

Features:<br>
1. one-on-one live chatting
2. user friendly
3. no login or sign up required
4. Supports multiple rooms at a single time
